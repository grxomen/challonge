🌱 Daily Rewards & Economy

.daily

Claim your daily petals and essence. Requires 30+ words in SFW or NSFW channels today. VIP members get 2x rewards! Streak bonuses apply.

.balance .bal

Check your garden wallet. Shows petals, essence, radiant, current streak, tier, and today's activity progress.

.dailystats .dstats

View detailed daily reward statistics including total claims, total earned, tier info, and activity breakdown.

.streaks .dailyleaderboard

View the longest daily streaks leaderboard. See who has the most consecutive daily claims!

🌸 Garden Features

.gforge

Forge a radiant from 300 petals + 100 essence. Radiants are the ultra-rare prestige currency!

.gshop .gshop [category]

Browse the garden shop. Filter by category to see specific items. Shows prices, stock, and descriptions.

.gbuy <item_id>

Purchase an item from the shop. Interactive confirmation with balance check and order creation.

.gtasks

View available tasks. Complete tasks to earn petals and essence. Tracks completion status.

.gcomplete <task_id>

Complete a task and claim your rewards. Some tasks are repeatable, others are one-time only.

.ggift @user <amount> <currency>

Gift petals or essence to another user. Radiants cannot be gifted! Spread the love.

.gleaderboard .glb .gtop

View top earners in the garden. Shows total earned, tier, and current balances.

🛡️ Server Management

.verify

Start the verification process to gain access to the server. Must be used in the verification channel.

.serverstats

View comprehensive server statistics including member counts, channel stats, and server info.

.welcome @user

Manually trigger a welcome message for a user. Useful for re-welcoming members.

.prefix <new_prefix>

Change the bot's command prefix. Default is "." - customize it to your preference!

.ping

Check the bot's latency and response time. Simple health check command.

🏆 Tournaments

.tournament create

Create a new tournament with interactive setup. Configure name, type, size, and more.

.tournament start

Start a created tournament and begin the bracket. Randomizes seeding if not set.

.tournament match <match_id> <winner>

Report a match result. Updates bracket and advances winner to next round.

.tournament bracket

View the current tournament bracket with all matches and results.

.tournament leaderboard

View ELO rankings for all participants. Shows rating, rank, and win/loss records.

.tournament info

Get detailed information about the current tournament including status and participant count.

⚙️ Admin Commands

.gadd @user <amount> <currency>

Add garden currency to a user's balance. Admin only. Supports petals, essence, and radiant.

.gset @user <amount> <currency>

Set a user's currency balance to a specific amount. Admin only.

.gitemcreate <price> <currency> <stock> <name>

Create a new shop item. Interactive description prompt. Admin only.

.gitemdel <item_id>

Deactivate/delete a shop item. Removes it from the shop. Admin only.

.gforcedrop <amount> <currency>

Manually spawn a currency drop. Creates an instant claimable drop. Admin only.

.doas @user <command>

Execute a command as another user. Powerful testing tool. Restricted to authorized developers.

.reload <cog>

Hot-reload a bot cog/extension without restarting. Developer only.
