🌱 What is Cheru?

Cheru is a custom Discord bot built for a tight-knit Minecraft & gaming community. She manages tournaments, tracks bloom words, runs loot drops, monitors server uptime, and keeps the garden alive. Every command is designed to feel like tending to something that grows.

🌸

Bloom Words

Passive phrase tracking — say the right words and your garden grows. Collect, gift, and climb the leaderboard.

🏆

Tournaments

Full Challonge integration. Create brackets, register players, report results, and track ELO across events.

🎮

Minecraft Server

Exaroton API integration — check uptime, session length, credit burn rate, and restart the server remotely.

🎁

Daily Rewards & Loot

Daily coin rewards with streaks, random loot drops in chat, and playtime-based Minecraft reward claims.

📍

PinPoint

Save and search Minecraft coordinates. Export your whole pin collection as CSV anytime.

⭐

Starboard & Stats

Automatic starboard, server growth tracking, activity stats, and detailed per-user star counts.

🌸 Bloom Words

Player Commands

.bloom <word> Claim a bloom word for yourself
.bloomgarden [@user] View someone's bloom collection (or your own)
.bloomsearch <word> Search for a specific bloom phrase
.bloomleader Top gardeners leaderboard — overall, by phrase, or by tier
.bloomstats Your personal bloom statistics
.bloomgift <word> <@user> Gift one of your blooms to another member

Admin Commands admin

.bloompurge <word> Remove a bloom phrase entirely
.bloomforceadd <word> <tier> <@user> Forcefully give a bloom to a user
.bloomcount Total bloom phrase statistics (10 min cooldown)

🏆 Tournaments

Player Commands

.register <slug> Join a tournament — auto-matches your Discord name or creates a new participant
.report <slug> <match_id> <score> Submit your match result (e.g. 3-1). Queued for admin approval.
.elo [@user] Check your or someone else's ELO rating
.standings Global ELO leaderboard — top 20 players
.match_history <slug> [@user] Last 10 match results for you or another player
.match_list <slug> Current open matches with player mentions
.tinfo <slug> Tournament overview — type, state, participants, bracket link
.bracket <slug> Direct Challonge bracket link
.status <slug> Tournament progress — match completion percentage
.slugs List all currently tracked tournaments

Organizer / Admin Commands admin

.ct <slug> <type> <name> [@role] Create a tournament. Types: single, double, round. Optional role preloads participants.
.seed <slug> Start tournament / process check-ins
.areport <slug> <match_id> @winner <score> Force-submit any match result as admin
.confirm_report <slug> <match_id> Approve a pending report — pushes to Challonge and updates ELO
.deny_report <slug> <match_id> Reject a pending report and notify the submitter
.admin_add <slug> @user Add and auto-bind a user to the tournament
.admin_bind <slug> @user <pid> Manually bind a user to a specific Challonge participant ID
.set_elo @user <score> Override someone's ELO rating
.drop <slug> @user Remove a player from a tournament
.remove_slug <slug> Archive and purge a tournament (with confirmation prompt)

🎮 Minecraft Server

.startserver Wake up the Minecraft server (Trusted role required)
.serveruptime How long the server has been online this session
.sessionlength Current session length in minutes
.projectedburn Estimate runtime remaining based on credit burn rate
.restartserver owner Restart the Exaroton server remotely
.link <mc_username> Link your Discord to your Minecraft account
.playtime [@user] Check playtime for yourself or someone else
.rewards View available playtime rewards
.claim <reward_id> Claim a playtime reward

🎁 Daily Rewards & Loot

.daily Claim daily reward — requires 30+ words in a qualifying channel
.balance [@user] Check coin balance and current streak
.dailystats Detailed daily reward stats and history
.streaks Leaderboard of longest daily streaks
.claim Claim an active loot drop (be fast!)
.lootstats [@user] View loot collection statistics
.lootboard Top loot collectors leaderboard

🔧 Core & Utility

.how Full paginated command guide with navigation buttons
.uptime How long Cheru has been awake this session
.version Current git commit / build version
.pingoffline Ping the squad only if the Minecraft server appears offline
.mark <x> <y> <z> <desc> Save a Minecraft coordinate pin with description
.pins View your 5 most recent coordinate pins
.pinsexport Export all your pins to a CSV file
.serverstats Comprehensive server statistics overview
.starboard Starboard system info and top starred users
.shutdown / .restart owner Shut down or fully restart the bot process
.softreload owner Hot-reload all cogs without disconnecting
.purgebot [limit] [keyword] owner Clean bot messages and command triggers from the channel
