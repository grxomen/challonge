🔒 Cheru Bot - Privacy Policy

Last Updated: March 18, 2026

Summary: Cheru Bot only collects data necessary to provide its services within Discord. We do not sell or share your data with third parties.

1. Information We Collect

Automatically Collected Data:

  • Discord User ID: Your unique Discord identifier
  • Username: Your Discord username (for display purposes)
  • Server ID: The Discord server where the Bot is used
  • Message Activity: Count of messages sent (for statistics, NOT message content)
  • Join/Leave Events: When members join or leave the server

User-Generated Data:

  • Economy Data: Virtual currency balances (Petals, Essence, Radiant)
  • Transactions: Records of virtual currency exchanges and purchases
  • Tasks: Completion records for Bot tasks
  • Daily Rewards: Claim history and streak information

2. How We Use Your Information

We use collected data to:

  • Provide Bot functionality (economy, rewards, statistics)
  • Track virtual currency balances and transactions
  • Generate server statistics and activity reports
  • Prevent abuse and maintain system integrity
  • Improve Bot features and user experience

3. What We Do NOT Collect

  • ❌ Message content (we only count messages, not read them)
  • ❌ Private/Direct messages
  • ❌ Email addresses or phone numbers
  • ❌ Payment information (no real money transactions)
  • ❌ IP addresses or location data

4. Data Storage

Your data is stored in:

  • Local SQLite databases on our secure server
  • JSON files for configuration and statistics
  • Data is encrypted in transit
  • Regular backups are maintained for data integrity

5. Data Sharing

We do NOT sell, rent, or share your data with third parties.

Data is only shared when:

  • Required by law or legal process
  • Necessary to prevent harm or fraud
  • You explicitly request data export

6. Data Retention

  • Active Data: Retained while you use the Bot
  • Deleted Conversations: Removed from memory nightly
  • Statistics: Aggregated data retained indefinitely
  • Economy Data: Retained unless server admin requests deletion

7. Your Rights

You have the right to:

  • Access: Request a copy of your data
  • Deletion: Request deletion of your data (contact server admins)
  • Correction: Request correction of inaccurate data
  • Opt-Out: Stop using the Bot to prevent future data collection

8. Children's Privacy

The Bot is intended for users 13+ (per Discord's Terms of Service). We do not knowingly collect data from children under 13. If we discover such data, it will be deleted.

9. Third-Party Services

The Bot may integrate with:

  • Discord API: For all Bot functionality
  • Exaroton API: For Minecraft server status (if configured)

These services have their own privacy policies.

10. Security

We implement security measures including:

  • Secure server infrastructure
  • Regular security updates
  • Access controls and authentication
  • Encrypted data transmission

However, no system is 100% secure. Use the Bot at your own risk.

11. Changes to This Policy

We may update this Privacy Policy. Continued use after changes constitutes acceptance. Major changes will be announced in servers where the Bot is active.

12. Contact Us

For privacy questions or data requests, contact the Bot administrators in the Discord server where Cheru is active.

13. Discord's Privacy Policy

This Bot operates on Discord's platform. Please review Discord's Privacy Policy for information on how Discord handles your data.
